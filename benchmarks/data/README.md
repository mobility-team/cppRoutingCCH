# DIMACS Florida road network

`USA-road-t.FLA.gr.gz` is the Florida travel-time graph from the 9th DIMACS
Implementation Challenge. It contains 1,070,376 nodes and 2,712,798 directed
arcs. The compressed file is 14,807,799 bytes.

- Source: <https://www.diag.uniroma1.it/~challenge9/data/USA-road-t/USA-road-t.FLA.gr.gz>
- Official download page: <https://www.diag.uniroma1.it/~challenge9/download.shtml>
- SHA-256: `A316684184E78276C40828CDB2B5C098B315A9EB2F71A8F02E6220973AF87658`

The official download page states that the challenge data files are in the
public domain unless otherwise noted. It also documents known gaps and errors
in the road data. This graph is included only as a reproducible scaling
benchmark, not as an authoritative representation of Florida roads.
